Cassandra Labs Overview

This page summarizes each hands-on lab, what it demonstrates, and how it maps to the theory topics.

Lab 01: Keyspace Basics

  • Keyspace creation, SimpleStrategy, basic table
  • Maps to: NoSQL basics, partitioning, replication

Lab 02: Partitioning and Clustering

  • Partition keys, clustering columns, bounding partitions
  • Maps to: Partitioning, clustering, data modeling

Lab 03: Modeling by Query

  • Query-driven modeling, denormalization, lookup tables
  • Maps to: Query-first design, denormalization

Lab 04: Indexes and Materialized Views

  • Secondary indexes, materialized views, trade-offs
  • Maps to: Indexes, materialized views

Lab 05: Consistency, LWT, and Batching

  • Consistency levels, lightweight transactions, batches
  • Maps to: Consistency, LWT, batching

Lab 06: TTL and Tombstones

  • TTL, tombstones, deletes, compaction
  • Maps to: TTL, tombstones, deletes

Lab 07: Aggregation and Filtering

  • Aggregation, ALLOW FILTERING, counters
  • Maps to: Aggregation, filtering, counters

Lab 08: Relational to Query-First

  • Converting relational models to query-first Cassandra models
  • Maps to: Data modeling, denormalization

Lab 09: Multi-DC Replication (Placeholder)

  • NetworkTopologyStrategy, multi-DC setup
  • Maps to: Multi-DC replication, advanced topics

Lab 10: Security Basics (Placeholder)

  • Authentication, roles, permissions
  • Maps to: Security features

Lab 11: Monitoring and Repair (Placeholder)

  • nodetool, cluster health, repairs
  • Maps to: Monitoring, repair
